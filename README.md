WeatherWave 🌤️

A responsive weather dashboard built with vanilla JavaScript, Bootstrap, and OpenWeatherMap API. Search any city for current conditions + 7-day forecast with smooth dynamic updates.
✨ Features
Real-time Weather: Current temperature, wind, and conditions for any city

7-Day Forecast: Hourly/daily predictions with weather icons

Mobile-First Design: Bootstrap responsive grid + Font Awesome icons

Smooth UX: Instant search with loading states and error handling

Clean UI: Modern cards, gradients, and intuitive layout

🛠️ Tech Stack 
Frontend: HTML5, CSS3, JavaScript (ES6+)
Framework: Bootstrap 5
Icons: Font Awesome 6
API: OpenWeatherMap API (Current + Forecast endpoints)
Deployment: GitHub Pages

🚀 Quick Demo
Enter city name (e.g., "Pune", "Mumbai", "Latur")

See current weather instantly

🔧 Setup & Run Locally
# Clone repo
git clone https://github.com/Sumitd11/WeatherWave.git
cd WeatherWave

# Get free OpenWeather API key: https://openweathermap.org/api
# Add your key to script.js (line with 'YOUR_API_KEY')

# Open index.html in browser
live-server  # or use VS Code Live Server extension
